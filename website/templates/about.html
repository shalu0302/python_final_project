Data Set Information

Data set link: england 2020-2021 Football Match

Variables Information:

  • Round: This column usually represents the number of round for which the data is collected or reported. It can include information such as the number of round which the player is participating.
  • Date: Date stands for the Date in which player participated in the match.
  • Team 1: This colum show the Name of the player participated in Team 1.
  • FT: FT shows the score of the match.
  • Team 2 : This column typically indicates the name of player in Team 2.
